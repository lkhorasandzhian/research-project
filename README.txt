-------------------|Bank marketing strategy data analysis and web-scaping of nike.com|--------------------
-------------------------------------------------- -------------------------------------------------------------
Authors: https://github.com/evelyn-lab & https://github.com/lkhorasandzhian
This folder contains files of 4 formats: csv, pptx, xlsx, ipynb. The csv file contains the initial data from the 
kaggle.com website at the link https://www.kaggle.com/datasets/berkayalan/bank-marketing-data-set . 
The pptx file contains just information and conclusions based on the information received. In the meantime, 
the xlsx file serves as the uploaded result of the second part of the web scraping work in the ipynb file. Finally, 
the ipynb file contains the source code for both data manipulation from kaggle.com and web scraping from 
nike.com.
